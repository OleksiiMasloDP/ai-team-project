# Roles

## Адам — Frontend

Задачи:

- Внешний вид интерфейса
- Структура страниц
- Подключение стилей и бутстраппа (через npm)
- Адаптивная версия

---

## Али — Server

Задачи:

- Настройка сервера
- Обработка frontend запросов (связь index.js с server.js)
- Возврат данных в формате JSON

---

## Константин — AI

Задачи:

- Подключение AI API (Cohere)
- Создание функции запроса к AI (общение с AI)
- Обработка ответа AI
- Передача результата в backend (базу)

---

## Мурсал — База данных (MongoDB)

Задачи:

- Подключение MongoDB
- Создание схем
- Сохранение данных
- Получение данных из базы
